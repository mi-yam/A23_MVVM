		- プレビューの再生・一時停止
		- 最初のクリップから再生

		- †プレビューのスキップ

		- 拡大縮小
		- トリミング
		- カット（分割）
		- エフェクト（フェードイン フェードアウト 時間経過）
	- タイムラインについて
		- 動画ファイルの追加（クリップ）
		- 画像ファイルの追加
		- テンプレートの追加
		- 音声等の追加
		- 比較画面の追加
		- 説明画面の追加
	- マニュアル作りについて
		- テンプレート機能

		- 字幕・テロップの追加
		- 説明画面の追加
		- 画面分割での同時再生（良い例と悪い例）
